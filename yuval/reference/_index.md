# Yuval / Reference — Index

תמונות השראה לסגנון הבית. יובל סורק את התיקייה הזו בתחילת כל משימת יצירת תמונה ומחלץ פלטה, קומפוזיציה, אווירה — ומשלב את הלקחים ב-prompt שהוא בונה.

## הנחיות הפקדה

- דרופ קבצי `.png` / `.jpg` / `.jpeg` / `.webp` כאן. אין צורך לתעד טקסטואלית — יובל קורא את התמונות עצמן.
- שמות slug-friendly: `<מילה-מתארת>-<מספר>.png` (לדוגמה: `interior-cafe-01.jpg`).
- 3–5 תמונות מספיק להתחלה — לא כמות, איכות.
